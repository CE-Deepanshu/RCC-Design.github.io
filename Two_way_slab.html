Slab Design — Step-by-step Procedure

Compact guide for two-way slab design: aspect ratio, depth, effective span, loads, moments and reinforcement sizing.

1 Step 1 — Aspect ratio

Compute aspect ratio:

AR = ly / lx
  • For two-way slab behaviour the usual requirement is AR < 2 (i.e., ly / lx < 2).
  • If AR ≈ 2 confirm panel type from the relevant code or clause — panels near the limit may need special verification.

2 Step 2 — Overall and effective depth

Method A — Start from assumed overall depth

  • Assume overall slab depth D (mm).
  • Assume clear cover c (mm) and choose main bar diameter φ (mm). Typical φ: 8, 10, 12, 16 mm.
  • Effective cover: cover_eff = c + φ / 2
  • Effective depth: d = D − cover_eff = D − c − φ/2
  • All lengths are in mm unless stated otherwise.

Method B — Empirical starting depth

  • Use empirical relation: d = Span / (B.V × M.F)
  • B.V = base value (from code clause or table — e.g., code clause 23.2.1 or your reference table).
  • M.F = modification factor (typical range 1.0–2.0; commonly 1.5).
  • Round/adjust d to the nearest practical bar arrangement and verify with Method A if needed.

3 Step 3 — Effective span (Leff)

Choose effective span definition based on boundary conditions and code guidance:

  • Simply supported panels: L_eff ≈ clear span + d (use l + d).
  • Continuous panels / supports bearing both sides: L_eff = clear span + (sum of bearing widths) / 2, where sum of bearing widths = support_width1 + support_width2.
  • Always use the effective span definition specified by your design code for the panel type in question.

4 Step 4 — Design loads (per unit area)

Use consistent units: w in kN/m² for panel loads; convert to N/mm² or N/mm for sectional checks when needed.

Dead load (DL)

  • Self-weight of slab = γconcrete × thickness.
  • Typical γconcrete = 25 kN/m³. So self-weight = 25 × (thickness in m) kN/m².
  • Add superimposed dead loads (finishes, screeds, fixed equipment) to get Total DL (kN/m²).

Live load (LL)

  • Include imposed occupancy loads and other variable loads.
  • Total live load = LL (kN/m²).

Design (factored) load

Combine DL and LL using the appropriate factors from your code to obtain w (kN/m²) for ULS (ultimate limit state).

5 Step 5 — Design moments for the panel

Use bending coefficients (αx, αy) for a rectangular panel supported on four sides from code tables (e.g., Table 26 or equivalent).

Mx = αx × w × lx2
My = αy × w × ly2
  • Units: w in kN/m² and l in m → M in kN·m per meter width. Convert to N·mm if using working formulas.
  • Negative moments at continuous supports → design top reinforcement.
  • Positive moments at mid-span → design bottom (main) reinforcement.

6 Step 6 — Main reinforcement design (per meter width)

Use the limit-state bending equation in consistent units. A commonly used working formula:

Ast = (0.5 × fck / fy) × [1 − √(1 − (4.6 × M) / (fck × b × d²))] × b × d

Where:

  • Ast = required steel area (mm²) per width b (use b = 1000 mm for per-meter design).
  • M = bending moment (N·mm) per meter width (convert kN·m/m → N·mm by ×106).
  • fck = characteristic compressive strength of concrete (N/mm²).
  • fy = yield strength of steel (N/mm²).
  • b = width considered (mm) (use 1000 mm for per-meter).
  • d = effective depth (mm).

Bar selection and spacing

Abar = (π / 4) × φ²   (mm²)
no. bars per m = Ast / Abar
spacing s (mm) = 1000 / (no. bars per m) = (Abar × 1000) / Ast

Spacing limit: maximum spacing between main bars shall not exceed min(3 × d, 300 mm) according to typical code clause (e.g., cl. 26.3.3(b)(1)). Verify clause numbers and local amendments.

7 Step 7 — Torsion / corner reinforcement

  • If torsion reinforcement is required by code (corner panels with torsion), follow the relevant code clauses (for example D-1.8 to D-1.11 or your code references).
  • Working rule (approximate): Ast_torsion ≈ (3/4) × Ast_max where Ast_max is the maximum main steel required.
  • Provide torsion reinforcement at extreme corners (not along continuous edges), typically within 1 m from the corner and up to 1 m length.
  • Choose torsion bar diameter (e.g., 8–10 mm) and compute Abar and spacing as done for main bars.
