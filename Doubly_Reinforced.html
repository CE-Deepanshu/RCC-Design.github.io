Simply Supported Beam

Doubly Reinforced Section – Design Steps

Given Data
Grade of Concrete = M__ , fck = ___ N/mm²
Grade of Steel = Fe__ , fy = ___ N/mm²
Overall depth of section, D = ___ mm
Width of section, b = ___ mm
Step 1: Calculate Effective Depth (d)
1. Assume nominal clear cover (Table 16, Cl. 26.4.2)
Clear cover, c = ___ mm
2. Assume bar diameter = ___ mm (20mm if not given)
3. Effective cover = c + 0.5 × bar dia.
d = D − (effective cover)
Step 2: Calculate Effective Span (Leff)
1️⃣ Leff = Clear span + d
2️⃣ Leff = Clear span + (support width/2 + support width/2)

✅ Adopt minimum of (1) & (2)

If only clear span given:
Leff = Clear span + d

If c/c span given:
Leff = Centre-to-centre span
Step 3: Load Calculation
✅ Dead Load (DL):
Self weight = (b/1000) × (D/1000) × 1 × 25 = ___ kN/m
Additional load = ___ kN/m
WDL = ___ kN/m

✅ Live Load (LL):
WLL = ___ kN/m

✅ Factored Load:
Wu = 1.5 (WDL + WLL) = ___ kN/m
Step 4: Ultimate Forces
4.1 Ultimate Bending Moment (UDL only)
Mu = Wu × Leff² / 8 = ___ kNm

4.2 Ultimate Shear Force
Vu = Wu × Leff² / 2

4.3 Limiting Moment of Resistance (Mu,lim)
Refer Annex G, IS 456
Mu,lim = 0.36 fck b xu,max/d × (1 - 0.42 xu,max/d) × b d²
If Mu,lim < Mu → Doubly Reinforced Beam required
Step 5: Reinforcement Design
5.1 Compression Steel (Asc)
Mu − Mu,lim = fsc Asc (d − d')
⇒ Asc = (Mu − Mu,lim) / (fsc(d − d'))

Compute:
εsc = 0.0035 × (xu,max − d') / xu,max
Get fsc from IS 456 / SP 16

Select bars → compute number of bars

5.2 Tension Steel (Ast)
Ast = Ast1 + Ast2
Ast1 = 0.36 fck b xu,max / (0.87 fy)
Ast2 = Asc × fsc / (0.87 fy)

Minimum Steel Check:
Ast,min = 0.85 b d / fy
Step 6: Shear Design
Nominal shear stress:
τv = Vu / (b d)

Find τc,max from Table 20 (IS 456)
Find τc from Table 19 (Pt%)

If τv > τc → design shear reinforcement

Vus = Vu − τc b d
Assume stirrup → Asv

Sv = 0.87 fy Asv d / Vus

Limit checks:
Sv ≤ 0.75d
Sv ≤ 300 mm
Provide **minimum** of permitted spacings
